# Para rodar o projeto após realizar o CLONE
```
flutter clean
flutter pub cache repair (esse vai demorar um pouco)
flutter pub get
```
É preciso ter a versão >= 3.3.1 e < 4.0.0>, para isso use `flutter upgrade`
